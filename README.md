# Enigma Business Report Generator

Ultra-simple business analysis tool. Just enter a business name and get a complete report instantly.

## How to Use

1. Open `index.html` in any browser
2. Type a business name
3. Click "Generate Report"
4. Export to PDF

That's it!

## Features

- ✅ Single input field (just business name)
- ✅ Automatic industry detection
- ✅ Competitor analysis
- ✅ Market trends
- ✅ SEO keyword research
- ✅ Revenue opportunities
- ✅ Action plan
- ✅ PDF export (via print)
- ✅ No installation required
- ✅ Works offline
- ✅ 100% standalone HTML file

## Installation

None needed! Just download and open `index.html` in your browser.

Or clone this repo:

```bash
git clone https://github.com/jcaruso666-rgb/enigma-report-gen.git
cd enigma-report-gen
open index.html
```
